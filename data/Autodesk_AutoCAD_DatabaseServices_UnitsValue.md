# Autodesk.AutoCAD.DatabaseServices.UnitsValue Enumeration

## Overview

#### Description
This .NET class wraps [AcDb::UnitsValue](AcDb__UnitsValue.md) ObjectARX class. It indicates the type of units of length measure used by the drawing.
```text
public enum UnitsValue {
  Undefined,
  Inches,
  Feet,
  Miles,
  Millimeters,
  Centimeters,
  Meters,
  Kilometers,
  MicroInches,
  Mils,
  Yards,
  Angstroms,
  Nanometers,
  Microns,
  Decimeters,
  Dekameters,
  Hectometers,
  Gigameters,
  Astronomical,
  LightYears,
  Parsecs,
  USSurveyFeet,
  USSurveyInch,
  USSurveyYard,
  USSurveyMile
}
```
