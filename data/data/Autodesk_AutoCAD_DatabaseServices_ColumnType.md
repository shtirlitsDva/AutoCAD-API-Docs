# Autodesk.AutoCAD.DatabaseServices.ColumnType Enumeration

## Overview

#### Description
This .NET enum wraps the [AcDbMText::ColumnType](AcDbMText__ColumnType.md) ObjectARX enum. 
It defines the types of columns.
```text
public enum ColumnType {
  NoColumns,
  StaticColumns,
  DynamicColumns
}
```
