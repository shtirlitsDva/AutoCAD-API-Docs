# Autodesk.AutoCAD.DatabaseServices.DataType Enumeration

## Overview

#### Description
This .NET class wraps the DataType ObjectARX class.
```text
public enum DataType {
  Buffer = 0x80,
  Date = 8,
  Double = 2,
  General = 0x200,
  Long = 1,
  ObjectId = 0x40,
  Point = 0x10,
  Point3d = 0x20,
  Resbuf = 0x100,
  String = 4,
  Unknown = 0
}
```
